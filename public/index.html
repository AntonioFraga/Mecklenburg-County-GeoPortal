What are you looking for?

Try the first part of an address like 700 N, a place name or a parcel ID.

Created by Mecklenburg County GIS • Source code on Github

  1. Welcome!

    GeoPortal is designed to convey a host of information in an easy-to use manner. Click the Next button below to begin your tour!

  2. Search

    Use the Search box to find your location. Just type the first part of what you're looking for and let the search control do the work for you. As an example, try typing 700 n. Then use your keyboard or mouse to select your match.

    You can search for:

    • Addresses
    • Place names (parks, libraries, businesses and others)
    • Parcel ID's

  3. Map

    The Map is fully interactive - you can pan, zoom, and select locations by mouse or touch.

  4. Print, Embed, or Contact Us

    Use the top bar to Print a report, make Embed code to put GeoPortal on your own site, or Contact us.

  5. Get the Source Code

    Mecklenburg County makes the full source code to GeoPortal available on Github under a MIT open source license. Feel free to customize GeoPortal for your own municipality, and patches are always welcome!
